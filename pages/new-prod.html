Instrucciones para crear una base de datos

Para poder crear una base de datos en Inventify, tiene a disposición dos métodos:

  1. Puede cargar un archivo .xlsx (Excel) con los datos de sus productos y proveedores. El sistema leerá el nombre del archivo .xlsx y lo asignará como nombre de la base de datos. Se recomienda usar un nombre descriptivo, por ejemplo, producto.xlsx si desea crear una base de datos de productos.
  2. Otra opción es utilizar el modelo recomendado. Puede acceder al modelo desde este LINK, el cual puede ser modificado según las necesidades de su empresa.
  3. IMPORTANTE: Una vez que haya creado la base de datos, guarde el modelo .xlsx utilizado, ya que lo necesitará para realizar futuras modificaciones o agregar nuevos datos.
