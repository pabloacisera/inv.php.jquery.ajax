Crear y cargar base de datos

Usted cuenta con dos métodos para generar y completar los datos en una base de datos de los sus productos y proveedores.

  • Primero debe asegurar que ha creado, o existe una base de datos de productos. Para conocer si existe checkear base de datos
  • Una vez creada la base de datos, puede proceder a cargar los datos tanto de manera manual como a través de nuestro sistema automatizado.
  • Si escoge el sistema automatizado deberá, o bien, descargar nuestro modelo de Xlxs o cargar uno propio.

Si no ha creado una base de datos con el método anterior, aún así puede crear una de manera automática cargando directamente el archivo modelo Xlsx previamente proporcionado, y la base de datos se generará automáticamente. Es recomendable que dicho modelo se guarde en una carpeta segura en su equipo o se utilice el servicio proporcionado a tal fin, y esto es necesario porque si en algún momento quisiera modificar los datos, debe utilizarse el modelo con el cual se ha creado la base de datos.
